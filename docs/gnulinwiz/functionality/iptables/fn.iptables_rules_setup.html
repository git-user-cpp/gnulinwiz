pub fn iptables_rules_setup() -> i8
Expand description

Applies iptables rules from the system configuration file.

This function uses iptables-restore to immediately apply the rules stored in /etc/iptables/iptables.rules by executing a bash command with sudo privileges. It logs success or failure with appropriate messages.

  • 1 if applying the rules fails.
