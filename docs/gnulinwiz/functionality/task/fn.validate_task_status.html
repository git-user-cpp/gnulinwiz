pub fn validate_task_status(status: i8)
Expand description

Validates the status of a task and exits the program on failure.

Checks the provided status code and, if it is non-zero, prints a failure message using print_setup_status_failed and terminates the program with the status code as the exit code. This function ensures that any task failure halts the configuration process to prevent further errors.
