pub fn zram_swap_setup() -> i8
Expand description

Configures ZRAM swap by copying the ZRAM generator configuration file.

Copies the ZRAM configuration file from ../configs/zram-generator.conf to /etc/systemd/zram-generator.conf using cp with sudo privileges. This sets up compressed RAM-based swap to improve system performance. Logs success or failure with appropriate messages.

  • 0 if the configuration file is copied successfully.
  • 1 if the copy operation fails (e.g., due to permission issues or missing source file).
